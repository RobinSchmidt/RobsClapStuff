An example project using the C++ wrapper class around the clpa C-API from here:

  https://github.com/free-audio/clap-helpers/blob/main/include/clap/helpers/plugin.hh
  https://github.com/free-audio/clap-helpers/blob/main/include/clap/helpers/plugin.hxx

The files in the Source folder have the following origins:

dllmain.cpp  
was auto generated by Visual Studio when creating a new .dll project.

create-an-actual-plugin.cc
is from clap-helpers-main/tests/  and was modified (commented out the catch2 stuff, adjusted 
includes)

checking-level.hh
host-proxy.hh
host-proxy.hxx
misbehavior-handler.hh
plugin.hh
plugin.hxx
version-check.hh
are from clap-helpers-main/include/clap/helpers
